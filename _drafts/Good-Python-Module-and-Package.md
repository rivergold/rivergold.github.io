# time

## Convert `int` into time

```python
import time
x = 4000
time_str = time.strftime('%H:%,:%S, time.gmtime(x))
print(time_str)
>>> '01:06:40'
```

**_Ref:_** [stackoverflow: Is there anyway to convert those int into time?](https://stackoverflow.com/a/47871587/4636081)
