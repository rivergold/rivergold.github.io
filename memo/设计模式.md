# Vistor

访问者模式

**核心理解**

- 核心功能：将数据结构和在数据结构上的操作结偶
- vistor 中的每个类表示了一种操作
- 数据结构的 element 使用`accept`在于通过多态实现在不需要检查 element 类型的情况下，实现动态绑定到 visitor 对应于该 element 类型的操作
